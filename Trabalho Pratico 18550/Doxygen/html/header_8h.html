◆ OperationsList

typedef struct OperationsList OperationsList

Estrutura para armazenar as operações.

◆ OperationsMath

typedef struct OperationsMath OperationsMath

Estrutura utilizada para resultados em calculos com Operações.

Um OperationsMath contém o resultado dos calculos (@res) e o apontador para o lista de operações utilizadas para obter o resultado (@operations).
